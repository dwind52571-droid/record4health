# Daily Health Tracker

一个面向个人使用的轻量健康记录产品，用来记录日常饮食、体重和运动，并按天、周、月查看统计数据。

## 目录结构

- `docs/product`：产品方案、MVP 范围、页面结构
- `docs/technical`：数据模型、技术设计草案
- `docs/ux`：交互稿、文案、视觉方向
- `apps/web`：后续手机端 Web / PWA 代码
- `packages/shared`：后续共享类型、工具函数、常量
